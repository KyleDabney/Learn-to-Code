Don't Get Distracted

Keep It Simple!

  • Download Visual Studio Code
  1. w3schools.com Read through the material here for HTML only! Understand it before moving on to css
  2. Practice creating ugly websites with just HTML
  3. w3schools.com Read through the material here for CSS only! Understand it before choosing your language of study.
  4. Practice implementing CSS into your ugly HTML pages that you just created
  5. I recommend either JavaScript or Python for your first choice of language. Python will be easier to understand if you've never been involved with programming
  6. w3schools.com Read through the material here for JavaScript language only! Understand it before trying to implement this into an HTML/CSS project you've been working on

    7. OR...

  7. w3schools.com Read through the material here for Python language only! Understand it before trying to implement this into a project you've been working on
  8. The last step is to build a simple website, then build it again and again until you don't need to look up how to do it. Then build a more complex website and continue the same pattern as above. You'll always need to look something up as you get further into the craft. Once you fully understand what you're doing, then create your final website (Portfolio) which includes your more impressive works. finally, start working on interview questions and applying for jobs (100's of them) The more you apply, the more you'll understand.
